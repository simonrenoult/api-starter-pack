# API Starter Pack

> Bootstrap Express APIs faster.


## Commands

Start the application:
```sh
$ npm start
```

Start the application in watch mode:
```sh
$ npm run watch
```

Run the tests:
```sh
$ npm test
```

Run the tests in watch mode:
```sh
$ npm run test:watch
```
